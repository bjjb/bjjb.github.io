jjbuckley's Github Projects

Here are my current GitHub-hosted open-source projects. Please feel free to fork and improve anything you find even remotely useful.

These generally address some HTML5 tasks that I found myself having to repeat time and again.

Bailitheoir

A small JavaScript implementation of the W3's HTML5 form constraints specification.

Although some of the better browsers already have some form of constraints validation on HTML5 forms, none of them (right now) seem to have implemented it completely.

This small JavaScript library addresses some of the issues. The homepage has a small form which utilises the constraints.

FS (Feidhm Soghluaiseacht)

A boilerplate mobile web application. Write some standards compliant HTML5, and plug this in to have an application that runs on iPhone or Android phones.

The homepage itself uses the script, so viewing the source is probably the best way to see how it works.
